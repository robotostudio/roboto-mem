# Nothing enters Team Memory without human review

Every Entry — Standard or Lesson — lands via a PR that a teammate approves; there is no auto-share mode, and Personal Memory never syncs on its own. We chose this over direct-push or auto-sync (the faster "hive mind" options) because Entries are injected into every teammate's agent sessions automatically: one wrong or secret-bearing Lesson would propagate to the whole team before anyone read it. Trust in Team Memory is the product; review is what buys it. At org scale the gate is tiered — scope Owners via CODEOWNERS, with org-wide Standards needing two approvals from the standards group — and automated secret/PII scanning on every Promotion and in the repo's CI backs the human gate, because 200 writers outrun eyeball review.
